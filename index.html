🔗 ربط قاعدة البيانات — Supabase

ربط Supabase يخلي بيانات الزيارات تتزامن فوراً بين جميع أجهزة المندوبين والأدمن في الوقت الحقيقي

1
افتح supabase.com → أنشئ حساب مجاني → اضغط New Project
2
بعد إنشاء المشروع → اضغط Settings → API → انسخ الـ URL والـ anon key
3
الصق القيم في الحقول أدناه واضغط ربط الآن
